Capstone documents

  • Project Abstract - Similar to the project flyer, this document serves as a one-page summary of the project. The format for an abstract is very different, however. This project will require an abstract in the Spring 2025.

  • Presentation - to AGIE team 11/1/2014 - this is the presentation Alina gave to the AGIE team on Friday, Nov. 1, 2024.

Task 1 - Other GIE sites review

  • Other GIE sites - Task 1 - This manuscript contains a summary of other gender inclusion sites, answer questions like: when were they created? Why? Are they still active? What was learned? Did they create any rubrics or other artifacts relevant to our project?

Task 2 - NSF ADVANCE review

  • NSF ADVANCE Summary - Task 2 - This report that details the NSF advance program, the partners and papers, and other important features that can influence our direction. Who is responsible for overseeing this draft? ONE person should be owner and others should contribute.

Task 3 - NIH landing page review

  • NIH Landing Page Review - Task 3 - examples of NIH landing pages, what is required, what makesa good or bad landing page. Considerations and lessons learned for design of our landing page.

Task 4 - AGIE repository design

  • AGIE Repository Design - Task 4 - This document describes the AGIE repository design including user personas, information architecture, data structure, etc.

Task 5 - AGIE repository build

  • AGIE Repository Users Guide - Task 5 - This document provides an overview of the AGIE site, how it operates, and how to modify it.
